Overview

Built end-to-end,
from chip to report

A full-stack desktop application replicating the software layer of a real lab-on-chip diagnostic device — live sensors, protocol execution, and AI drug ranking.

🌡️

Live Environment Monitoring

Six sensors — temperature, humidity, CO₂, O₂, pressure and pH — stream live data from a cloud C++ backend as real-time charts.

⚗️

7-Step Protocol Engine

Guides the technician through the full microfluidic workflow: intake, dissociation, droplet generation, drug loading, incubation, imaging, and analysis.

🔬

Fluorescence Well Viewer

Visualises simulated microscopy frames per drug well with alive/dead cell overlays, efficacy scores, and a ranked top-5 treatment panel.

📄

Auto-generated PDF Report

On protocol completion, a signed clinical report exports to Downloads — patient info, step log, QC sign-off, and treatment recommendations.

🚀

Cloud-hosted Backends

Two simulation servers — C++ for sensors, Python for oncology — deployed on Railway. No local setup required.

⚙️

CI/CD Release Pipeline

GitHub Actions builds macOS and Windows releases on every version tag push, with automatic version injection and DMG packaging.

Workflow

From login
to clinical report

The app follows the exact workflow a lab technician would use — from authentication through to a signed PDF ready for physician review.

1

Authenticate as technician

The technician logs in through a secure screen. Only authorised personnel can access the diagnostic platform.

2

Select patient & cancer profile

Choose from a list of simulated patients with different cancer types and stages. Each session is scoped to a single patient.

3

Monitor the lab environment

Verify that temperature (37°C), CO₂ (5%), humidity, O₂, pressure and pH are all within target — streamed live from the C++ backend.

4

Run the 7-step protocol

Execute the full microfluidic workflow with real-time step timers, progress tracking, pause/resume, and abort controls.

5

Review oncology drug rankings

The Python backend ranks all 16 drug combinations by tumour cell kill rate, with fluorescence well images per drug.

6

Export PDF clinical report

A full clinical report is generated and saved to Downloads — patient data, step log, QC sign-off, and top recommended treatment.
